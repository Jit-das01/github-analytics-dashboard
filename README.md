# 🚀 GitHub Analytics Dashboard

A beautiful, real-time analytics dashboard for GitHub repositories built with React and the GitHub API.

![Dashboard](https://img.shields.io/badge/Status-Live-success)
![React](https://img.shields.io/badge/React-18-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## ✨ Features

- 📊 Real-time repository statistics (stars, forks, watchers, issues)
- 💻 Primary language detection
- 📦 Repository size analysis
- 🎨 Beautiful gradient UI design
- ⚡ Fast and responsive
- 🔗 Direct links to analyzed repositories

## 🛠️ Technologies Used

- **React 18** - UI framework
- **GitHub REST API** - Data fetching
- **Vanilla CSS** - Styling with gradients
- **No build tools** - Runs directly in browser

## 🚀 Try It Yourself

1. Download or clone this repository
2. Open `index.html` in your browser
3. Start analyzing repositories!

## 📖 How to Use

1. Enter a GitHub username (e.g., `facebook`)
2. Enter a repository name (e.g., `react`)
3. Click "Analyze Repository"
4. View comprehensive stats instantly!
